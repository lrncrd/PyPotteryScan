PyPotteryScan

Description

PyPotteryScan is an AI-powered tool for archaeological drawing processing. It uses state-of-the-art OCR technology (OlmOCR-7B) to automatically extract and annotate text from ceramic artifact drawings, streamlining the documentation workflow for archaeologists and researchers.

Features

  • ✓ Batch image processing with bounding box annotation
  • ✓ Advanced OCR with OlmOCR-7B (4-bit quantized)
  • ✓ Interactive drawing cleanup with eraser tool
  • ✓ Manual and automatic text extraction modes
  • ✓ Structured export with CSV metadata
